# AXIOM

A floating AI companion for macOS. Powered by Claude. Lives on your desktop and gets things done.

---

## What is AXIOM?

AXIOM is a always-on desktop assistant that sits quietly in the corner of your screen as a pixel art character. It understands context, controls your Mac, searches the web, reads your screen, and runs automations for you.

---

## Features

### Pixel Companion
- Animated pixel art character with distinct states — idle, thinking, talking, happy, error
- Borderless floating window, always on top, draggable anywhere
- Blinks, reacts, and animates in response to what it's doing

### Mac Control
- Open, close, and focus applications
- Control system volume and brightness
- Lock screen
- Window management — move, resize, snap
- Deep system automation via AppleScript and shell

### Context Awareness
- Read and write your clipboard
- Capture and OCR your screen to understand what's visible
- Detect which app is active and in focus

### Web
- Search the web and return plain language answers
- Open URLs directly in your browser

### AI Brain
- Powered by Claude (Haiku) for fast, smart responses
- Full tool-use loop — AXIOM picks the right tool automatically
- Persistent conversation history within a session
- Custom personality and system prompt

### Automations
- Define custom shortcut commands
- Chain multiple actions together
- Run shell scripts on demand
